# 总结：
 <br>1，传入费用不收费 
 <br>2，不同可用区间公有子网或者私有子网之间的传输有费用，使用公网IP或者弹性IP传入传出都有费用
 <br>3，区域间传输有费用
 <br>4，专有线路传输有费用，另外有端口的使用费用
